# $sub[]
Devuelve la resta de los números proporcionados.

**Sintaxis**
```
$sub[Número;...]
```

**Parámetros**
- `Número` `(Tipo: Entero, Flotante || Indicador: Obligatorio)`: Los números a restar. Separe los valores múltiples con `;`.

**Ejemplo**
```
Resultado: $sub[20;10;6]
```
