# $divide[]

Divide los números proporcionados.

**Sintaxis**
```
$divide[Números]
```

**Parámetros**

- `Números` `(Tipo: Entero, Flotante || Indicador: Obligatorio)`: Los números a dividir. Separe los números con `;`.

**Ejemplo**
```
Respuesta: $divide[10;$message[-1]]
```
