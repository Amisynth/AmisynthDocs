# $message[]

Devuelve el mensaje del usuario *(sin el disparador de comando)*.

**Sintaxis**
```
$message[Argumento]
```

- `Argumento`: `(Tipo: Numero || Marca: Obligatorio)`: Obtiene el numero de argumento del comando, -1 para devolver todo.

**Ejemplo**
```
$message[-1]
```
