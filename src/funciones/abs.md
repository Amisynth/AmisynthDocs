
# $abs[]
Devuelve el numero absoluto proporcionado.

## Sintaxis
```
$adbc[Número]
```

### Parámetros
- `Número` `(Tipo: Coma flotante, Entero || Indicador: Obligatorio)`: Devuelve el valor absoluto prporcionado.

### Ejemplo
```
$argsCheck[>2;❌ Uso no válido. Uso: `!promediar (número1)`]
$abs[$message[1]]
```
