# $getTextSplitLength[]

Devuelve la cantidad de fragmentos obtenidos después de dividir un texto con `$textSplit[]`.  

**Sintaxis**  
```
$getTextSplitLength
``` 


**Ejemplo #1**
```plaintext
$textSplit[hello%world%!;%]
> $getTextSplitLength   // Devuelve 3
```
