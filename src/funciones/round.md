# $round[]

Redondea hacia arriba el número proporcionado.

## Sintaxis
```
$round[Número;(Decimal)]
```

### Parámetros
- `Número` `(Tipo: Flotante || Indicador: Obligatorio)`: El número a redondear.

- `Decimal` `(Tipo: Entero || Indicador: Vacante)`: El número de decimales a redondear. El valor predeterminado es `0`.

### Ejemplo
```
Resultado: $round[100.123;1]
```
