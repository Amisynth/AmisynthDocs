# $suma
Devuelve la suma de los números proporcionados.

**Sintaxis**
```
$sum[Número;...]
```

**Parámetros**

- `Número` `(Tipo: Entero, Flotante || Indicador: Obligatorio)`: Los números a sumar. Separe los valores múltiples con `;`.

**Ejemplo**
```
Resultado: $sum[5;6;1]
```
